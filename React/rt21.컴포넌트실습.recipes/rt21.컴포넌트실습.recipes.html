npm 사용법 
        CRA(create-react-app) 를 사용해서 리액트 프로젝트를 만들고 recipes.html
        에 작성한 컴포넌트를 프로젝트에 적용하여 본다.
        * CRA 는 create react app 의 약자.

        # 프로젝트 생성
        $ npx create-react-app recipesprj01

        # 폴더 이동
        $ cd recipesprj01

        # package.json 파일이 있는지 확인
        $ ls

        # 리액트 프로젝트 서버 실행하기
        $ npm run start

        # src/App.css 수정

        # src/App.js 수정

        # src/Recipe.js 생성

        # src/Menu.js 생성
